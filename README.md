# Pois Art

Pois Art 是一个移动端优先的照片海报生成 Web 应用。当前版本已经收敛到一条更明确的产品路径：`首页上传一张主图 -> 编辑页生成双 panel 成品 -> 可选上传一张填充块照片 -> 调整波点与底板 -> 导出分享`。

更详细的实现状态见：

- [当前实现进展文档](./docs/IMPLEMENTATION_STATUS.md)
- [协作开发说明](./docs/COLLABORATION_NOTES.md)

## 当前产品定义

- 首页只负责上传 `主照片`
- 编辑页不再提供顶部图片栏
- 画板固定是 `双 panel`
- 布局只支持两种切分方向：
  - `左右分块`
  - `上下分块`
- 单主图场景下：
  - 主照片按原始宽高比生成真实照片块
  - 另一侧自动生成 `1:1 对应尺寸` 的填充块
  - 如果没有额外上传填充块照片，就使用底板样式渲染
  - 如果上传了填充块照片，就用该照片替换右侧/下侧填充块

## 当前已完成能力

### 1. 主流程

- 首页上传主照片后自动进入编辑页
- 编辑页可以继续上传一张 `填充块照片`
- 支持高清导出、下载、系统分享和复制分享文案

### 2. 画板与交互

- 成品默认缩放到画板中央，约占可视区域 `50%`
- 支持拖动画布平移
- 支持滚轮缩放
- 支持移动端单指拖动和双指缩放
- 双击可回到默认 fit 状态
- 当前只对 `主照片 panel` 提供裁切交互

### 3. 布局面板

- 仅保留 `左右分块 / 上下分块`
- 不再暴露画幅、主 panel 占比、留白、块间距等参数

### 4. 填充块面板

- 支持上传 `填充块照片`
- 保留底板样式：
  - `纯色底`
  - `横条纹`
  - `像素底`
- 保留颜色与纹理参数：
  - `主色`
  - `辅色`
  - `条纹粗细`

### 5. 波点面板

- 形状保留：
  - `五角星`
  - `水滴`
  - `雪花`
  - `圆形`
- 点位分布改为：
  - `随机分布`
  - `单侧波点`
  - `双侧波点`
- 保留可调参数：
  - `点大小`
  - `大小差异`
  - `总点数`
  - `装饰点`
  - `透明度`
  - `大小变化`
- 水滴和雪花形状路径已重画，视觉上更接近真实目标

## 当前限制

- 当前主路径只支持 `一张主照片 + 一张可选填充块照片`
- 填充块照片暂时没有单独的裁切 UI
- 还没有撤销 / 重做
- 还没有系统化自动化测试覆盖

```bash

npm run preview
```

## 当前重点

当前版本的重点不是继续扩展更多模板，而是把这条单图主路径做准：

- 默认成品尺寸与视口表现稳定
- 主图与填充块严格拼接
- 波点交换逻辑一致
- 交互和导出结果尽量接近参考站的直接感
