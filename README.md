# Pois Art

Pois Art 是一个移动端优先的照片海报生成 Web 应用。它会把用户上传的照片转成“上下双区”的海报：上半区保留主图，下半区生成条纹或纯色底板，并叠加可配置的形状采样波点。

## 当前阶段

当前项目处于 `Alpha+ / Beta 准备阶段`。

这意味着：

- 工程已经从空仓库搭建完成并可构建
- 主流程已经打通：上传 -> 自动生成预览 -> 调整参数 -> 随机 -> 高清导出 -> 分享/下载
- 主题模板、Canvas 渲染引擎、Worker 导出链路都已经落地
- 第二阶段的关键增强已经接入：真实裁切、画幅预设、更多主题、导出格式
- 还没有进入真实设备上的细致视觉校准和完整测试阶段

更详细的进展说明见：

- [当前实现进展文档](./docs/IMPLEMENTATION_STATUS.md)

- 已完成一次本地构建验证：`npm run build` 成功

### 2. 页面与产品结构

- 已实现首页：

  - 主题模板展示

### 4. 主题模板

当前内置 10 套主题：

- 星星海报
- 雨滴海报
- 雪花海报
- 圆点海报
- 像素夜景
- 落日拼贴
- 薄荷静物
- 糖霜云朵
- 纸感墨点
- 青柠胶片

每套主题已绑定：

- 默认上下分区比例
- 默认底板颜色与风格
- 默认形状与数量
- 默认视觉气质

### 5. 导出与分享

- 预览使用 Canvas 2D 实时渲染
- 高清导出优先走 `Worker + OffscreenCanvas`
- 不支持时回退到主线程导出
- 已支持导出格式切换：`PNG / JPEG`
- 分享优先走 `Web Share API`
- 不支持时可下载 PNG，并支持复制分享文案
- 已支持本地保存最近一次项目参数草稿

## 项目结构

```text
src/
  App.tsx                     应用状态与主流程
  components/
    HomeScreen.tsx            首页
    EditorScreen.tsx          编辑页
    ExportSheet.tsx           导出结果浮层
  render/
    engine.ts                 海报渲染主引擎
    worker.ts                 导出 Worker
    workerClient.ts           Worker 调用封装
    shapes.ts                 形状路径生成
    random.ts                 随机与基础数学工具
  presets.ts                  主题模板与默认参数
  types.ts                    核心类型定义
  styles.css                  全局样式与移动端适配
public/
  manifest.webmanifest        PWA manifest
  sw.js                       service worker
  icon.svg                    应用图标
```
